# Social Media API
 
This is a REST API where Users can register and log in using Bearer (JWT) tokens. They will be able to create posts, leave comments, and like existing posts.

I have utilized Pydantic library for Data and Schema Validation.

There is a 100% test coverage where I have utilized PyTest library to write integration and unit tests for the API.

The application is deployed on render, you can access it [here](https://social-media-api-r1sh.onrender.com/docs).
